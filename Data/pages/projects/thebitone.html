Pitch:
I participated in the BIT1 Game Dev Competition to challenge myself with a tight deadline while testing how much I could accomplish solo. The result is "Damnorak" a 3D roguelite heavily inspired by The Binding of Isaac. While admittedly a sloppy version of its inspiration, it was a valuable learning experience where I experimented with advanced systems like procedural room generation and AI-assisted development.

What i did:
• Developed a 3D roguelite game loop with procedural elements
• Built procedural room generation system (with AI assistance)
• Experimented with LLMs for voice and code generation

What I Learned:
• Delegates in C#
• Object-oriented programming principles and SOLID fundamentals
• Unity's NavMesh system for AI navigation
• Shader graph basics
• The risks of AI-assisted coding without fully understanding the implementation, the AI-generated room generation system became deeply tangled in the project and would be extremely difficult to extract as a reusable Unity package

Final Reflection:
This project taught me valuable lessons about both rapid prototyping and the limitations of AI assistance in development. While I learned a lot technically, the biggest takeaway was understanding how AI-assisted code can create technical debt if you don't fully grasp what it's doing. The tight deadline forced me to prioritize features and make tough decisions about scope, but relying too heavily on AI for complex systems like procedural generation created maintainability issues. Despite the challenges and the "sloppy" execution, I had fun and gained crucial insights that will inform my future development practices.
