BIT1 GAME DEV COMPETITION 2025


"Damnorak" was developed solo in 1.5 months for the BIT1 Game Dev Competition, focusing on core gameplay mechanics and architectural foundations.

Key technical achievements:
• Implemented event-driven architecture for scalable system communication
• Applied OOP principles for maintainable and extensible code structure
• Developed procedural room generation algorithm for dynamic level creation
• Integrated Unity NavMesh for AI pathfinding and character navigation
• Created custom VFX systems for immersive gameplay feedback

This project taught me the critical balance between rapid prototyping and sustainable development practices. Learning to allocate time effectively between planning, implementation, and iteration was the most valuable takeaway from this intensive solo development experience.
