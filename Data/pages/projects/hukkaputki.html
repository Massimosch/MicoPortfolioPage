Pitch:
Hukkaputki is an educational text-based adventure game developed for youth education, tackling sensitive topics like substance abuse and mental health through interactive storytelling.

What i did:
As one of the core developers, I built this interactive visual novel using vanilla JavaScript with a JSON-driven narrative system with one other developer, I did mostly write JavaScript, HTML and CSS.

Key Features:
• Dynamic choice based storytelling with branching narratives
• JSON data structure for efficient story content management
• Responsive CSS design for optimal cross device experience

What I Learned:
• Structuring narrative flows
• Designing data driven system by separating content from code using JSON
• Collaborating with stakeholders to address sensitive topics responsibly
• Project planning and collaboration using Miro, Trello, Git, and Figma

Final Reflection:
This project deepened my understanding of JavaScript fundamentals and taught me how to create maintainable, data-driven applications. The JSON-based architecture made it easy for non-technical team members to update story content without touching the code. Working on an educational game with real social impact was both challenging and rewarding.
