Hukkaputki



Hukkaputki is an educational text-based adventure game developed for youth education.

As one of the developers, I built this interactive visual novel using vanilla JavaScript with a JSON-driven narrative system.

The game features:

• Dynamic choice-based storytelling with branching narratives
• JSON data structure for efficient content management
• Responsive CSS design for optimal cross-device experience

The project challenged me to create an engaging educational tool that addresses sensitive topics like substance abuse and mental health through interactive decision-making.
